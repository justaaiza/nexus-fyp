# Nexus FYP

**Nexus FYP** is a modern, role-based frontend dashboard designed to streamline the Final Year Project (FYP) management process at FAST NUCES. 

Built with React, Vite, and Tailwind CSS, this platform provides a centralized hub for all FYP stakeholders to collaborate effectively and track progress throughout the academic year. The interface features a clean, responsive dark-mode aesthetic with specialized domains for four distinct roles:

- **Students:** Submit proposals, track milestone deadlines, upload deliverables, and view rubric-based grading feedback.
- **Supervisors:** Manage student requests, create group milestones, and evaluate submissions using interactive digital rubrics.
- **Coordinators (Admin):** Broadcast global announcements, manage user registrations, and orchestrate jury panel assignments.
- **Jury Members:** Access assigned project repositories, review submitted documents, and input final defense scores.

## Tech Stack
- **Framework:** React 18 & Vite
- **Styling:** Tailwind CSS v4 using a custom CSS variable design system
- **Routing:** React Router v7
- **Icons:** Lucide React

## Development
To run this project locally:

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
